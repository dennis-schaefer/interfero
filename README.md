## Building from Source
To build Interfero from Source run the following maven command:
```shell
mvn -pProduction clean package
```
This will create a runnable JAR file in the `target` directory named `interfero-<version>.jar`. This JAR file contains
the whole application with frontend and backend as well as all dependencies.
